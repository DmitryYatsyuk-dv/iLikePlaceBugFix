# iLikePlace
Application based on ***UITableView***

Use frameworks: 
  **UIKit** ; 
  **MapKit** ;
  **CosmosStar** ;
  **Realm** .
  
Includes:
```         
Work with: images, cells, annotations ; 
Sorting ; 
DataBase(Add/Edit/Save/Delete) ;
Tracking User Location & Get directions to place .

 ```
